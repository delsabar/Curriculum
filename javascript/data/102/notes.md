# JavaScript Data 102
## Three simple data types

### string
Strings are defined with either `'` or `"`.
```
let location = 'Atlanta, Georgia';
```

### boolean
The value of a boolean is either `true` or `false`.
```
const isNew = false;
```

### number
Is a number. Can do decimals. Has a cap.
```
let age = 32;
```
